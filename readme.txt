ReadMe-Project 1
Nicholas Croulet


Html file with JS scripts and CSS styling to create a online travel booking site that uses JSON to dynamically create destination pages.
Has home page with cards to represent each destination along with links to destination pages that have more details on the location.

Has a form for user to enter their personal details in order to book a trip. Form alerts user when submission suceeds. Has information
on several destinations including an itinerary and a map using leaflet. Leaflet, W3schools, and previous projects were used.
 Html/CSS and JS were used as well. Has a search function to load specific pages.
